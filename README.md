# RFM Analysis using Python & SQL

## Project Overview
This project performs RFM (Recency, Frequency, Monetary) analysis on customer transaction data
to segment customers into categories such as VIP, Loyal, Regular, New, and Lost.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (MySQL)
- Git & GitHub

## Project Structure
- Python: Data cleaning, RFM calculation, visualization
- SQL: RFM calculation using window functions and segmentation
- Dataset: Transaction data used for analysis

## RFM Segments
- VIP
- Loyal
- Regular
- New
- Lost

## Outcome
Customer segmentation insights to support marketing and retention strategies.
